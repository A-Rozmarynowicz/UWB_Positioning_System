\doxysubsection{Detailed Description}
Timer management module for the Lighthouse system. 

This module provides initialization and control of hardware timers used for:\+
\begin{DoxyItemize}
\item UWB activation delays
\item ACK message timeouts
\end{DoxyItemize}

The timers are implemented using ESP32 hardware timers and are used across different state machine transitions.
