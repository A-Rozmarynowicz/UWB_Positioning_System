\doxysubsection{Detailed Description}
Global configuration header for the Lighthouse system. 

This file contains system-\/wide configuration constants, hardware pin definitions, and shared includes required by multiple modules of the Lighthouse firmware.

It defines:\+
\begin{DoxyItemize}
\item Global system parameters (number of lighthouses, observer ID)
\item GPIO pin assignments used to determine the lighthouse identifier
\item Common framework and module dependencies
\item Global variables shared across compilation units 
\end{DoxyItemize}
