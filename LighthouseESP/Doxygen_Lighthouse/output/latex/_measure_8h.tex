\doxysubsection{Detailed Description}
Distance measurement and position calculation utilities. 

This header defines data structures, constants, and shared variables used for distance measurement processing and position estimation of Lighthouse devices within the system.

It provides:\+
\begin{DoxyItemize}
\item Limits and parameters related to distance measurements
\item A 3D position structure
\item Global buffers storing distances and calculated positions 
\end{DoxyItemize}
