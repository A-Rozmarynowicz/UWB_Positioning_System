Detailed Description

Distance measurement and position calculation utilities.

This header defines data structures, constants, and shared variables used for distance measurement processing and position estimation of Lighthouse devices within the system.

It provides:

  • Limits and parameters related to distance measurements
  • A 3D position structure
  • Global buffers storing distances and calculated positions
