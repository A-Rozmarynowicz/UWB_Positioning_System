Detailed Description

Hardware interface abstraction for LEDs and user button.

This header defines the hardware interface used by the Lighthouse system, including status LEDs and a user button. It provides constants describing GPIO assignments and declares public and private interface-related functions.

The interface layer is responsible for:

  • Initializing GPIOs used for LEDs and button input
  • Visual indication of system status and data transfer activity
  • Handling button interrupts
