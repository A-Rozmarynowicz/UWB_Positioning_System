\doxysubsection{Detailed Description}
LCD display handling and transmission queue utilities. 

This header defines constants, types, and shared variables used to manage an LCD display over I2C, including a software queue for asynchronous transmission.

It provides:\+
\begin{DoxyItemize}
\item LCD command and data queue management
\item LCD initialization and cursor positioning
\item String printing functions
\item Callback functions for I2C transmission handling 
\end{DoxyItemize}
