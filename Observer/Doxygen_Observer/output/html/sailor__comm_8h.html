Detailed Description

Communication interface for the sailor device.

This header defines constants and data layout used for sending information from the sailor device to the observer (or other nodes).

It provides:

  • Message size constant
  • Data layout enum for encoding sailor messages
  • Public function declarations for initialization and sending position
