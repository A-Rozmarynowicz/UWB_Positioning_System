Detailed Description

UWB (DW1000) initialization and management.

This header contains constants, external variables and function declarations used to manage the DW1000 UWB module. It provides functions for initialization, update loop handling, enabling/disabling the module, and UWB callbacks.

◆ UWB_ADDRESS_LENGTH

#define UWB_ADDRESS_LENGTH   8

Length of a UWB address in bytes.

UWB uses 64-bit addresses (8 bytes). This constant is used to define address arrays and to perform address comparisons.
