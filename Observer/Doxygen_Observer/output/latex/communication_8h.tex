\doxysubsection{Detailed Description}
ESP-\/\+NOW communication interface and message definitions. 

This header defines the communication protocol used by the system, including message formats, command identifiers, error codes, and helper functions for sending and receiving ESP-\/\+NOW packets.

It provides:\+
\begin{DoxyItemize}
\item Data layout and command enums for ESP-\/\+NOW messages
\item Global buffers and state structures for acknowledgements
\item Initialization and message sending utilities 
\end{DoxyItemize}
