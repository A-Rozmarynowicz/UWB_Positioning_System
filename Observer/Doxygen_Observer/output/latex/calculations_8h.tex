\doxysubsection{Detailed Description}
\doxylink{struct_position}{Position} estimation and matrix calculations for the observer. 

This header defines constants, global variables, and function declarations used to estimate the position of the observer device based on UWB distance measurements to multiple lighthouses.

It provides:\+
\begin{DoxyItemize}
\item Measurement limits and constants
\item Storage for distance measurements and lighthouse positions
\item Matrices and vectors for least-\/squares position estimation
\item Public API for updating distances and calculating position 
\end{DoxyItemize}
