\doxysubsection{Detailed Description}
Global configuration and shared type definitions. 

This header defines global constants, enums, and shared structures used across the project. It also includes all other module headers to ensure consistent access to configuration and interfaces.

It provides:\+
\begin{DoxyItemize}
\item Global system constants (e.\+g., number of lighthouses, device ID)
\item Shared data structures (e.\+g., \doxylink{struct_position}{Position})
\item Common enums used by multiple modules
\item Includes for all other project headers 
\end{DoxyItemize}
