\doxysubsection{Detailed Description}
UWB (Ultra Wide Band) initialization and handling utilities. 

This module provides configuration and helper functions for DW1000-\/based UWB communication. It includes initialization, enabling/\+disabling, address conversion and callback handlers for UWB events.
