\doxysubsection{Detailed Description}
ESP-\/\+NOW based communication interface for Lighthouse devices. 

This file defines the communication protocol, data layout, commands, and message handling functions used for wireless communication between Lighthouse units and observers using ESP-\/\+NOW.

It includes:\+
\begin{DoxyItemize}
\item Fixed-\/size data buffers and protocol constants
\item Enumerations describing message layout and command types
\item Error codes related to communication failures
\item Structures used to track ACK message state
\item High-\/level message sending API
\item Internal (private) callbacks and helper functions 
\end{DoxyItemize}
