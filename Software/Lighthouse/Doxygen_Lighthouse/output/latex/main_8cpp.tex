\doxysubsection{Detailed Description}
Initialization and main control program for a UWB Lighthouse. 

\begin{DoxyAuthor}{Author}
Arkadiusz Rozmarynowicz 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
01.\+2026
\end{DoxyDate}
This program is designed for an ESP32 microcontroller acting as a UWB lighthouse device. It is responsible for:\+
\begin{DoxyItemize}
\item Reading the lighthouse identifier (LIGHTHOUSE\+\_\+\+ID) from three configuration pins configured as INPUT\+\_\+\+PULLDOWN.
\item Initializing hardware interfaces, timers, communication modules, and the UWB subsystem.
\item Resetting and initializing the internal state machine controlling the lighthouse operation.
\item Disabling Wi\+Fi power-\/saving modes.
\item Periodically updating the UWB subsystem in the main program loop.
\end{DoxyItemize}

The application is built using the Arduino framework for ESP32.
