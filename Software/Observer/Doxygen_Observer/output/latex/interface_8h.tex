\doxysubsection{Detailed Description}
Interface utilities for LEDs and basic hardware feedback. 

This header defines constants and public functions used to control the onboard LEDs for status indication.

It provides:\+
\begin{DoxyItemize}
\item Pin assignments for UWB exchange and error indication LEDs
\item Initialization and control functions for the LEDs
