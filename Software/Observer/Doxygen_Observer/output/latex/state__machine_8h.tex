\doxysubsection{Detailed Description}
State machine interface for the observer. 

This header defines the state machine for the observer device. It includes:\+
\begin{DoxyItemize}
\item Definitions of possible states
\item Error types for state transitions
\item Global state data and current state tracking
\item Declarations of public state machine API functions
\item Declarations of state-\/specific handlers (enter, exit, callbacks) 
\end{DoxyItemize}
