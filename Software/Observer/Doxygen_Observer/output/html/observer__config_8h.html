Detailed Description

Global configuration and shared type definitions.

This header defines global constants, enums, and shared structures used across the project. It also includes all other module headers to ensure consistent access to configuration and interfaces.

It provides:

  • Global system constants (e.g., number of lighthouses, device ID)
  • Shared data structures (e.g., Position)
  • Common enums used by multiple modules
  • Includes for all other project headers
