Detailed Description

Observer communication module header.

This file defines the interface for communicating with the observer module via UART. It contains configuration constants, data structures, global variables, public API functions, and private helper functions and callbacks.

◆ _read_position()

void _read_position ( )

Reads and updates current position from UART buffer.

This function copies received position data from the UART buffer and adjusts it relative to the initial position. The first received non-zero position is stored as the initial reference.
